# 🏠 우리집 스마트 허브 (Our Family Smart Hub)

> **가계부와 편의점 관리를 하나로!**  
> 우리 가족의 스마트한 경제 생활과 자산 관리를 위한 프리미엄 스마트 홈 대시보드 솔루션입니다.

## ✨ 주요 기능

### 1. 💵 프리미엄 가계부
*   **실시간 수입/지출 관리**: Supabase Cloud DB를 통한 완벽한 실시간 데이터 동기화.
*   **저축 분리 로직**: 일반 지출과 저축을 분리하여 실질적인 순소비와 누적 저축액을 별도로 관리.
*   **이월 및 누적 계산**: 지난 달의 잔액과 전체 기간의 총 저축액을 자동으로 합산하여 표시.
*   **스마트 분석 메시지**: 현재 재정 상태를 분석하여 위트 있고 유용한 금융 조언 제공.

### 2. 🏪 우리집 편의점 (창고 관리)
*   **실시간 재고 추적**: 매대(창고) 내 물품의 수량을 자릿수별 슬라이딩 애니메이션으로 시각화.
*   **자동 소비 주기 시스템**: 설정한 기간마다 수량이 자동으로 차감되는 스마트 엔진 탑재.
*   **부족 알림**: 설정한 기준 수량 이하로 떨어질 경우 시각적 경고 및 브라우저 푸시 알림 제공.
*   **위트 있는 상태 메시지**: 물품 종류와 양에 따라 변하는 재치 있는 살림 메시지.

### 3. 📱 완벽한 모바일 최적화 (PWA)
*   **App-like Experience**: 스마트폰 홈 화면에 추가하여 네이티브 앱처럼 사용 가능.
*   **다크 모드 지원**: 사용자 선호에 따른 세련된 라이트/다크 테마 전환.
*   **오프라인 대응**: 서비스 워커(Service Worker)를 통한 향상된 성능 및 안정성.

## 🛠 기술 스택

*   **Frontend**: HTML5, Vanilla CSS (Premium Glassmorphism), JavaScript (ES6+)
*   **Backend**: Supabase (PostgreSQL, Realtime, Auth)
*   **PWA**: Service Worker, Web App Manifest
*   **Iconography**: FontAwesome 6.4.0, Google Fonts (Pretendard)

## 🚀 시작하기

1.  본 리포지토리를 클론합니다.
    ```bash
    git clone https://github.com/your-username/smart-home-hub.git
    ```
2.  `index.html`을 브라우저에서 엽니다 (혹은 GitHub Pages로 배포).
3.  Supabase 설정이 필요할 경우 `index.html` 및 각 JS 파일의 API Key를 본인의 설정으로 변경합니다.
